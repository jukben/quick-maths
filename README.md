Do quick maffs like BIG SHAQ! 


## Install

`yarn add quick-maths`

## Usage

```js
import Maffs from "quick-maths";

const quickMaffs = new Maffs(2).plus(2).is() // 4
```

### API

### `new quickMaffs(number: value)`

#### `.plus(number: value)`
#### `.minus(number: value)`
#### `.is()`
